# Repositório de Disciplinas

Repositório para armazenar labs e trabalhos das disciplinas cursadas na faculdade.

## Disciplinas

- **EDA** (Estruturas de Dados Avançados) — implementações e exercícios sobre estruturas de dados e algoritmos.
- **SO** (Sistemas Operacionais) — labs e projetos sobre processos, memória, concorrência e demais tópicos da disciplina.
